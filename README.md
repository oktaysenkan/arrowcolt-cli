# arrowcolt-cli

Command line helper for React Native.

[![asciicast](https://asciinema.org/a/0eBpDfA3seX3xjMB4y7P7IORL.svg)](https://asciinema.org/a/0eBpDfA3seX3xjMB4y7P7IORL)

## Installation

```bash
$ npm i arrowcolt-cli -g
```

## Usage

```bash
$ arrowcolt
```

```bash
$ npx arrowcolt
```
